Ouroboros manages the requirements, execution, evaluation, and records of AI coding work.

Your coding agent (Claude Code, Codex) writes the code. Ouroboros pins the spec before the run and verifies the result after it.

Install

Run in your terminal:

claude plugin marketplace add Q00/ouroboros
claude plugin install ouroboros@ouroboros

Requires Claude Code. No Python needed. Prefer a standalone CLI? pip install ouroboros-ai — see the install guide.
